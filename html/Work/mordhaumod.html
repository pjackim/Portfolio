Hardpoint Gamemode

Hardpoint is a game mode mod for a game named Mordhau and was created using Unreal Engine 4. This project was beneficial as it exposed me to unfiltered community feedback. Additionally, it yielded practice in understanding feedback -- when to incorporate and when to dismiss.

Critical takeaways from this project:

  • Processing feedback
  • Unreal Engine 4
  • Server & client replication
  • Test driven development
  • Iterative design
  • Communication with clients

Objective

The purpose of the game mode is to battle for positioning and occupation of the point which moves around the map based on a timer.

Objective

Once one team has the occupation of the point, the object begins awarding points to players and the team as a whole. The first team to 150 points wins the game.

The map shown in these images is a proof of concept map I designed as the client requested I only make the mode itself. My source code was open-sourced on GitHub, and content creators have made various maps around my creation, which is always lovely to see.

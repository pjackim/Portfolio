Gameplay Package

The Gameplay package provided users with various gameplay modifications:

Features

  • AFK
    • Prevent kicking of inactive player
  • Crouch Cooldown
    • Instant crouch
  • Dodge Cooldown
    • Instant dodge
  • Enable Dodge
    • Apply the 'Dodge' skill to current class
  • FOV
    • Increase periphery vision, very popular
  • Move
    • Allow movement while emoting
  • Parry
    • Automatically parry enemy attacks
  • Skip Warmup
    • Allow players to move durning match start.
  • Super Sprint
    • Sprint faster
  • Turncap
    • Adjust turn restrictions durning combat
